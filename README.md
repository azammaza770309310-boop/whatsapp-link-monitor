# Help Requests Monitor v9 - SIMPLE

بوت سحب طلبات المساعدة الدراسية - بسيط جداً!

## كيف يعمل؟

1. تشارك رابط البوت مع أصدقائك: `t.me/Azzamntheer2026_bot`
2. يضغطون Start
3. يضيفون البوت لمجموعاتهم الدراسية كـ Admin
4. ✅ البوت يراقب تلقائياً!

## لا يحتاج المستخدم لـ:
- ❌ رقم هاتف
- ❌ كود تيليجرام
- ❌ StringSession
- ❌ Pydroid 3
- ❌ أي تثبيت

فقط: اضغط Start + أضفني لمجموعتك!

## الأوامر (في الدردشة الخاصة مع البوت):
- `/start` - رسالة الترحيب
- `/status` - حالة البوت
- `/help` - المساعدة
- `/groups` - قائمة المجموعات المراقَبة

## الأوامر (في القناة - للمالك):
- `/help` - قائمة الأوامر
- `/status` - حالة البوت
- `/scan_week` - مسح آخر 7 أيام
- `/scan_month` - مسح آخر 30 يوم
- `/scan_stop` - إيقاف المسح
- `/groups` - قائمة المجموعات

## النشر على Render

### Start Command
```
python monitor_v9.py
```

### متغيرات البيئة
- API_ID, API_HASH, BOT_TOKEN, CHANNEL_ID
- STARTUP_SCAN_DAYS=7 (مسح آخر 7 أيام للمجموعات الجديدة)
