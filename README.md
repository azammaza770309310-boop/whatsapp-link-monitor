# Help Requests Monitor v8

بوت سحب طلبات المساعدة الدراسية - تسجيل تفاعلي بدون تثبيت!

## المميزات الجديدة في v8

### 🎯 تسجيل تفاعلي بدون تثبيت!
المستخدمون الجدد لا يحتاجون تثبيت Pydroid 3 أو أي شيء.

1. يفتح البوت في تيليجرام
2. يرسل `/start`
3. يرسل `/login`
4. يرسل رقم هاتفه → يستقبل كود → يرسل الكود
5. ✅ تم! البوت يراقب مجموعاتهم

### الأوامر في الدردشة الخاصة مع البوت:
- `/start` - ترحيب
- `/login` - تسجيل الدخول بحسابك
- `/status` - حالتك
- `/cancel` - إلغاء العملية

### الأوامر في القناة:
- `/help` - قائمة الأوامر
- `/status` - حالة البوت
- `/scan_week` / `/scan_month` / `/scan_full` - مسح تاريخي
- `/scan_stop` - إيقاف المسح

## النشر على Render

### Start Command
```
python monitor_v8.py
```

### متغيرات البيئة
- API_ID, API_HASH, BOT_TOKEN, CHANNEL_ID
- USER_SESSION_STRING (للمالك - اختياري الآن)
- STARTUP_SCAN_DAYS=30

## كيف يعمل التسجيل التفاعلي

1. المستخدم يضغط على البوت: t.me/Azzamntheer2026_bot
2. يرسل `/start` → يرى الترحيب
3. يرسل `/login` → يطلب منه رقم هاتفه
4. يرسل رقم هاتفه (+967...) → البوت يرسل كود لتيليجرام
5. المستخدم يفتح تيليجرام، ينسخ الكود
6. يرسل الكود للبوت → البوت يحفظ الجلسة
7. ✅ البوت يبدأ مراقبة مجموعات المستخدم فوراً
8. 📚 يبدأ مسح آخر 30 يوم تلقائياً
